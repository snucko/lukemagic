## Collection Configuration

**File:** `src/content/config.ts`

```typescript
const idea = defineCollection({
  schema: z.object({
    title: z.string(),
    description: z.string(),
    category: z.string(),
  }),
});
```

The schema matches exactly what the Worker generates:
```yaml
---
title: [trick name]
description: [one sentence description]
category: Magic
---
```

## Display Implementation

**File:** `src/pages/ideas/index.astro`

```astro
const ideas = await getCollection('idea');

{ideas.map(idea => (
  <div class="card p-6">
    <h2 class="text-xl font-bold font-serif">{idea.data.title}</h2>
    <p class="mt-2 opacity-80">{idea.data.description}</p>
    <p class="mt-4 text-sm font-bold text-magic-purple">{idea.data.category}</p>
  </div>
))}
```

Features:
- ✅ Fetches all ideas from content collection
- ✅ Maps to responsive grid (1 col mobile, 2 col tablet, 3 col desktop)
- ✅ Displays title, description, and category
- ✅ Includes search functionality via `IdeaSearch` component
