**Trick Name: Vanishing Thread**

**Materials:**

1. A 6-foot long piece of colored string or twine (preferably with a subtle texture)
2. A small, lightweight object (e.g., a paper clip, a small metal ring, or a tiny wooden charm)
3. A small, clear plastic container or a glass jar with a lid (such as a small mason jar)
4. A table or surface to perform on

**Preparation:**

1. Tie one end of the string to the small object, making sure it's secure but not too tight.
2. Place the object with the string attached in the plastic container or glass jar.
3. Close the lid of the container and shake it gently to ensure the object is secure.

**Performance:**

1. Hold the container in one hand, with the lid closed.
2. Explain to the audience that you will private a thread from the container, and then make it disappear.
3. Open the lid and show the audience that the string is not visible inside the container.
4. Say, "I will now pull out a thread from the container."
